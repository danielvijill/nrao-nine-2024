#fits folder

These are files containing information and data taken by the Very Large Array of NGC5258, a galaxy which is one of two interacting spiral galaxies in a system named Arp 240

NGC5258_s_concat_uvt_dot75asec_rbeam1asec_csdot1.pbcor_hcorr is a 2D raster image of the galaxy
NGC_5258_cube_1arcsec_inregion.pbcor_hcorr is a data cube which contains more information on the image
